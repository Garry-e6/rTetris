## rTetris

**rTetris** is a classic 2D Tetris game developed in **C++** using the **SFML** multimedia framework. The project was created to explore game programming fundamentals, including real time input handling, collision detection, piece rotation, line clearing, score management, and simple user interface development.

The game features a lightweight custom UI system built directly with SFML, including reusable **Button** and **TextBox** components. Player names and high scores are stored using **Enola**, my own C++ library for reading and writing INI configuration files.

This release includes the executable, required dependencies, and a **res** folder containing the game's fonts and audio assets. The background music was also composed specifically for this project.

### Features

* Classic Tetris gameplay
* Score tracking with persistent high scores
* Player name entry
* Custom Button and TextBox UI components
* INI-based save system powered by the Enola library
* Original background music
* Written entirely in modern C++

This project served as both a programming exercise and a foundation for experimenting with reusable game engine components and custom libraries.
