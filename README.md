# rTetris
**rTetris** is a classic 2D Tetris game built in modern C++ with SFML. It features custom Button and TextBox UI components, persistent high scores and player names using my Enola INI library, original background music, and classic gameplay. **Controls:** ←→ Move, ↑ Rotate, ↓ Soft Drop, Space Hard Drop, Esc Pause/Quit.
